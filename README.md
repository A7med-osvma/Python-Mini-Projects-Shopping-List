# Python Shopping List App 🛒🐍

A simple and interactive command-line application to manage a shopping list, built with Python. This is a great beginner-friendly project to practice using Lists, Functions, and While Loops.

## 🌟 Features
* **Add Items:** Easily add new items to your shopping cart.
* **Remove Items:** Delete items securely (includes error handling to check if the item exists).
* **View List:** Display all current items in your list.
* **Count Items:** Quickly check the total number of items you need to buy.
* **Interactive Menu:** A user-friendly loop that keeps the program running until you choose to exit.

## 🚀 How to Run
1. Make sure you have Python installed on your computer.
2. Download or clone this repository.
3. Open your terminal or command prompt.
4. Run the script using the following command:
   `python shopping_list.py`
